cara main game 2048

* WAJIB masuk ke terminal untuk menjalankan game 2048 ini. (game tidak bisa di jalan jika hanya menekan tombol run)

1. Gunakan tombol:
- W untuk menggeser ke atas
- A untuk menggeser ke kiri
- S untuk menggeser ke bawah
- D untuk menggeser ke kanan

2. Setiap geseran akan memindahkan semua angka ke arah tersebut.

3. Jika dua angka yang sama bertemu, mereka akan bergabung menjadi angka yang lebih besar (contoh 2 bertemu 2 menjadi 4).

4. Setelah setiap gerakan, satu angka baru (biasanya 2) akan muncul di tempat acak.

5. Tujuan permainan adalah mengumpulkan angka setinggi mungkin.

6. Game akan selesai jika board sudah penuh dan tidak ada angka yang bisa digabung lagi.

7. Tekan Q kapan saja untuk exit dari game.

8. Jika game over silahkan tekan tombol mana saja untuk keluar.
